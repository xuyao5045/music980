# music980
## 项目介绍
  这是一个基于Vue3的音乐播放器项目。

## 项目部署
- Node.js 版本：16.x 或以上
- MySQL 版本：5.7 或以上
### 1.前端部署
- npm install
### 2.后端部署
- cd backend
- npm install
### 配置数据库
- 修改配置：修改数据库连接信息，编辑 backend/config/config.js 文件

## 启动服务
### 1.前端服务
- npm run dev
### 2.后端服务
- node app.js
